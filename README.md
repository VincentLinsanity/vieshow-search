# Pre Requirement
- Nodejs
- Gulp

# Quick Start by Default
git clone project
```bash
$ git clone https://github.com/VincentLinsanity/vieshow-search.git
```

initial node_module
```bash
$ npm install
```

run client
```bash
$ npm install gulp -g
$ gulp
```

run server
```bash
$ npm start
```
